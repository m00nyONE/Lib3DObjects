- overview:

Lib3DObjects allows addon developers to create and manage 3D objects within The Elder Scrolls Online. It provides a robust framework for rendering controls in 3D space, making it easier to create immersive and interactive experiences.


- description:

Lib3DObjects is a comprehensive library designed for addon developers in The Elder Scrolls Online. It enables the creation and management of 3D objects, allowing for enhanced visual representation and interaction within the game world.
With Lib3DObjects, developers can easily render various types of 3D objects, including markers, lines, and textures, in both world space and render space. The library provides a range of features such as object pooling, animation support, and camera helpers to facilitate the development process.
The library is modular, with a clear separation between core functionalities, renderers, animations, and object types. This structure allows developers to extend and customize the library to suit their specific needs.
Lib3DObjects also includes a set of examples demonstrating its capabilities, making it easier for developers to understand how to utilize the library effectively in their addons.
Overall, Lib3DObjects is a powerful tool for addon developers looking to enhance their addons with 3D visual elements, providing a solid foundation for creating engaging and dynamic experiences in The Elder Scrolls Online.
